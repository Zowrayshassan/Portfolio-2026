# 🚀 3D Interactive Portfolio

This repository contains the source code for my interactive 3D portfolio website. Built with React, Three.js (React Three Fiber), and GSAP, it features a fully 3D interactive avatar, physics-based elements, and smooth scroll animations.

---

## ✨ Highlights

- **Interactive 3D Avatar**: Complex 3D models integrated natively in the browser using React Three Fiber.
- **Scroll & Storytelling**: Dynamic timelines and scroll-triggered visual sequences powered by GSAP.
- **Physics Integration**: Real-time physical collisions and interactions utilizing Rapier & Cannon.
- **Fully Responsive**: Optimized to provide a polished, smooth framerate experience across desktop and mobile devices.
- **Performance Focused**: Built rapidly with Vite and explicitly typed with TypeScript to ensure type-safety and fewer rendering bugs.

---

## 🛠️ Technologies Used

- **Framework**: [React.js 18](https://react.dev/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **3D Render Engine**: [Three.js](https://threejs.org/)
- **React 3D Integrations**: [@react-three/fiber](https://docs.pmnd.rs/react-three-fiber/), [@react-three/drei](https://github.com/pmndrs/drei)
- **Physics Engine**: [@react-three/rapier](https://github.com/pmndrs/react-three-rapier), [@react-three/cannon](https://github.com/pmndrs/use-cannon)
- **Animations**: [GSAP](https://gsap.com/)
- **Styling**: Vanilla CSS / Utilities
- **Analytics**: Vercel Analytics

## 🚀 Running Locally

To run this project on your local machine, follow these steps:

1. **Clone the repository**
   ```bash
   git clone <your-repo-link>
   cd Portfolio-Website
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```
   *(Note: This project uses `npm` as the package manager).*

3. **Start the development server**
   ```bash
   npm run dev
   ```
   This will run Vite locally, typically accessible at `http://localhost:5173/`.

4. **Build for production**
   ```bash
   npm run build
   ```
